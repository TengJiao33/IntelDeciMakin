# IntelDeciMakin

Research workspace for the adaptive preference-structure learning line in multi-criteria decision making.

This repository contains:

- experiment scripts for APSS proof-of-concept runs;
- processed benchmark datasets and manifest files;
- experiment outputs and notes;
- literature-reading notes and contribution-framing documents;
- vendored academic-research skills used as the project workflow guide.

Local-only material is intentionally excluded from the public repository, including remote-compute operational notes, raw downloads, full-text PDFs, recording transcripts, credentials, and generated caches.
